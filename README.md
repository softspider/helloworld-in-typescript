# HelloWorld on TypeScript

## SoftSpiders Tags

* ts
* hw

## Ancestors

No

## Requirements

* [Node.js](https://nodejs.org/en/download/package-manager/)
* [TypeScript](https://www.typescriptlang.org/)

## Istallation

Install TypeScript as a Node.js package

```sh
npm i -g typescript
```

## Compilation

```sh
tsc index.ts
```

## Running

```sh
node index.js
```
