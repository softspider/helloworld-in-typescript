## Requirements

* [Node.js](https://nodejs.org/en/download/package-manager/)
* [TypeScript](https://www.typescriptlang.org/)

## Istallation

Install TypeScript as a Node.js package

```sh
npm i -g typescript
```
or

```sh
yarn global add typescript
```

## Compilation

```sh
tsc index.ts
```

## Running

```sh
node index.js
```

or

```sh
yarn index.js
```
